Tom's Lab - pear repository

Registering the channel: 
pear channel-discover tomslabs.github.com/pear
Listing available packages: 
pear remote-list -c tomslabs
Installing a package: 
pear install tomslabs/package_name
Installing a specific version/stability: 
pear install tomslabs/package_name-1.0.0
pear install tomslabs/package_name-beta
Receiving updates via a feed: 
http://tomslabs.github.com/pear/feed.xml
