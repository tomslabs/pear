Guibod's pear repository

Registering the channel: 
pear channel-discover guibod.github.com/pear
Listing available packages: 
pear remote-list -c guibod
Installing a package: 
pear install guibod/package_name
Installing a specific version/stability: 
pear install guibod/package_name-1.0.0
pear install guibod/package_name-beta
Receiving updates via a feed: 
http://guibod.github.com/pear/feed.xml
